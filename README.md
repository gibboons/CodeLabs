# Android Basics with Compose - Codelabs

This repository contains my solutions to the Android Basics with Compose course codelabs from Google's Android Developer training.

### Unit 1 - Your First Android App

#### 1. [Greeting Card](./GreetingCard)
**Course:** [Unit 1 - Pathway 3](https://developer.android.com/courses/pathways/android-basics-compose-unit-1-pathway-3)

A simple greeting card app introducing Jetpack Compose basics.

**Interesting Aspects:**
- First introduction to Jetpack Compose and its declarative way of building UI.
- Learned how @Composable functions work and how UI is built from simple building blocks.
- Used basic layout elements like Column, Row, and Text.
- Preview mode made experimenting much faster.

#### 2. [Business Card](./BusinessCard)
**Project:** [Create a Business Card app](https://developer.android.com/courses/android-basics-compose/unit-2)

A professional business card application displaying contact information.

**Interesting Aspects:**
- Practiced combining Box, Column, and Row to build a clean layout.
- Worked with images and icons as part of the UI.
- Learned how spacing and padding affect visual structure.

#### 3. [Lemonade App](./LemonadeApp)
**Course:** [Unit 1 - Pathway 3](https://developer.android.com/courses/pathways/android-basics-compose-unit-1-pathway-3)

An interactive app simulating the process of making lemonade.

**Interesting Aspects:**
- First real use of state with remember and mutableStateOf.
- Built a simple multi-step interaction driven by state changes.
- Used conditional UI to switch screens based on current state.
- Learned the basics of  separating logic from UI.

### Unit 2 - Building App UI

#### 4. [Dice Roller](./DiceRoller)
**Course:** [Unit 2](https://developer.android.com/courses/android-basics-compose/unit-2)

A dice rolling app that generates random numbers with button interactions.

**Interesting Aspects:**
- A practical example of how state triggers recomposition.
- Combined button interactions with dynamic image updates.
- Helped understand how Compose updates only what needs to change.

#### 5. [Art Space](./ArtSpace)
**Course:** [Unit 2 - Interacting with UI and state](https://developer.android.com/courses/android-basics-compose/unit-2)

A virtual art gallery app for browsing artwork with navigation controls.

**Interesting Aspects:**
- Managed multiple pieces of state that change together.
- Added previous/next navigation through a small list of items.
- Combined images, text, and buttons into a clean layout.
- Handled edge cases when navigating the first or last item.

### Unit 3 - Display Lists and Use Material Design

#### 6. [30 Days App](./30DaysApp)
**Course:** [Unit 3](https://developer.android.com/courses/android-basics-compose/unit-3)

A motivational app displaying daily tips or activities over 30 days.

**Interesting Aspects:**
- First experience with LazyColumn for list rendering.
- Worked with data classes and larger sets of content.
- Used Material 3 components like cards.
- Added simple animations for expanding items.

## Technologies Used

- **Kotlin** - Primary programming language
- **Jetpack Compose** - Modern Android UI toolkit
- **Android Studio** - Development environment


## Running the Project

To run any of these projects:

1. Clone this repository
2. Open the project folder in Android Studio
3. Wait for Gradle sync to complete
4. Run the app on an emulator or physical device

## Reference

These projects are based on Google's official [Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course) course.
